它原本属于我的个人应用「纯纯写作」的独家技术之一，得益于我的付费分享频道朋友们的支持，这个项目才决定开源。

但也不要紧，我们的核心竞争力不仅仅于此，只是想通过以上内容 表明这是一个大家的付出和支持换来的开源，它是等价的。
